## ❗️ Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| API key not working        | Check your DeepSeek API key validity          |
| Slow responses             | Use a lower context length or faster model    |
| Antivirus warning          | Add the folder to exclusions                  |
| UI looks blurry            | Enable DPI scaling override in compatibility  |
